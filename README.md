## Problem Statement
Recruiters manually screen resumes, which is time-consuming and inconsistent.

## Solution
An NLP-based system that compares a resume with a job description and outputs:
- Match percentage
- Missing skills

## Input
- Resume (PDF)
- Job Description (Text)

## Output
- Similarity score
- Skill gap list

## Limitations
- Keyword-based matching
- Does not understand deep semantics
